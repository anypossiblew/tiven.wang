React 可以直接引入 HTML 页面里然后使用, 也可以使用 React CLI 创建一个全新的 React 项目. 本文讲解使用 an integrated toolchain for the best user and developer experience.

> Node >= 6 我的是 v10.15.3

```sh
npm install -g create-react-app
create-react-app my-app

cd my-app
yarn start
```

至此创建 React 成功.
